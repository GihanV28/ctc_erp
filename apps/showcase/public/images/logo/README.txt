LOGO PLACEMENT INSTRUCTIONS
============================

Please save your CCT logo image here with one of these names:
- logo.png (recommended)
- logo.svg
- logo.jpg

The logo should be the CCT logo image you provided.

Current expected filename: logo.png

After placing the logo, the application will automatically use it in:
- Login page
- Signup page
- Password reset pages
- Dashboard header
- All authentication pages

Recommended dimensions: 1200x300px (or similar aspect ratio)
Format: PNG with transparent background (preferred) or SVG
